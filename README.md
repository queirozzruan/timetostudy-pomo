# timetostudy

Uma aplicação minimalista de timer Pomodoro com lista de tarefas integrada. Projetada para ajudar no gerenciamento de sessões de estudo e intervalos de foco sem distrações.

## Funcionalidades

- **Modos de Timer:** Intervalos pré-configurados para Foco (40min), Pausa Curta (10min) e Pausa Longa (20min).
- **Gerenciamento de Tarefas:** Lista to-do integrada para adicionar, concluir e excluir tarefas durante as sessões.
- **Temas:** Alternância entre modo claro e um modo escuro de alto contraste.
- **Display em Tempo Real:** Indicador de data e hora atual no cabeçalho.
- **Sincronização de Aba:** O título da aba do navegador é atualizado dinamicamente com o tempo restante.

## Tecnologias

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (Vanilla ES6+)

## Como Executar

Como este projeto utiliza Tailwind CSS via CDN e JavaScript puro, não é necessário nenhum processo de build ou instalação de pacotes.

1. Clone este repositório em sua máquina local.
2. Abra o arquivo `index.html` em qualquer navegador moderno.

## Estrutura do Projeto

- `index.html`: Estrutura principal e estilização da aplicação.
- `app.js`: Lógica da aplicação (Timer, manipulação do DOM, Lista de tarefas).
